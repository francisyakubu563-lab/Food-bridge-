How It Works

1. Deposit - Drop food + pay fee at any branch

2. Send - We generate a pickup code for your receiver

3. Collect - Receiver shows code + ID at their branch to collect food

Transfer Fee: ₦1,500 flat fee per deposit + ₦200 per kg | ₦200 per carton of noodles
